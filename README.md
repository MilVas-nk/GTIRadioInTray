## Описание

GTIRadioInTray - это Windows-приложение, которое позволяет слушать онлайн-радио GTI Radio прямо из системного трея. Приложение предоставляет удобный способ прослушивания trance музыки с возможностью выбора качества звука и отображением информации о текущей композиции.

Официальный сайт радио: [https://gtiradio.ru/](https://gtiradio.ru/)

## Особенности

- **Минималистичный интерфейс**: Приложение работает в системном трее, не занимая места на рабочем столе
- **Выбор качества звука**: Возможность выбора между высоким (hi), средним (middle) и низким (low) качеством потока
- **Отображение метаданных**: Приложение отображает информацию о текущей проигрываемой композиции (исполнитель и название)
- **Автоматический запуск**: Приложение автоматически начинает воспроизведение при запуске
- **Надежное воспроизведение**: Обработка ошибок воспроизведения и повторное подключение при сбоях

## Технологии

Проект создан с использованием:

- **Язык программирования**: C# 12.0
- **Платформа**: .NET 9.0
- **Графический интерфейс**: Windows Forms
- **Аудио воспроизведение**: MediaPlayer (WPF Media)
- **Сетевые запросы**: HttpClient
- **Парсинг метаданных**: Регулярные выражения и JSON парсер
- **Логирование**: Встроенная система логирования

## Установка и запуск

### Требования

- Windows 10 или выше
- .NET 9.0 Runtime

### Установка

1. Скачайте последнюю версию приложения из раздела [Releases](https://github.com/MilVas-nk/GTIRadioInTray/releases)
2. Запустите `GTIRadioInTray.exe`

## Использование

После запуска приложение появляется в системном трее. Щелкните правой кнопкой мыши по иконке приложения, чтобы открыть контекстное меню с возможностями:

- **hi**: Воспроизведение в высоком качестве
- **middle**: Воспроизведение в среднем качестве (по умолчанию)
- **low**: Воспроизведение в низком качестве
- **Выход**: Завершение работы приложения

Приложение автоматически отображает информацию о текущей композиции в подсказке иконки в системном трее.

## Архитектура

Проект использует следующую архитектуру:

- `TrayApplicationContext`: Основной класс приложения, управляющий иконкой в системном трее
- `RadioMetadataService`: Сервис для получения и обработки метаданных из потока радио
- `MediaPlayer`: Класс для воспроизведения аудио потока
- `ILogger`: Интерфейс для логирования с реализациями `ConsoleLogger` и `NullLogger`

## Функции

- **Воспроизведение радио**: Воспроизведение потокового аудио с сайта gtiradio.ru
- **Управление качеством**: Переключение между тремя уровнями качества звука
- **Отображение информации о треке**: Отображение исполнителя и названия текущей композиции
- **Обработка ошибок**: Восстановление после сбоев воспроизведения
- **Мониторинг метаданных**: Постоянный мониторинг и обновление информации о текущей композиции

## Управление

Приложение полностью управляется через контекстное меню в системном трее. Никаких дополнительных окон не открывается.
